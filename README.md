##  Project Description
This project contains solutions for **Activities 1 to 10**, each placed inside its own file with its own class.  
Every activity has a **static void run()** method, and all activities are called from the **Main.java** file.

##  How to Run the Program

1. Open the project in your Java IDE (VSCode, BlueJ, etc.)
2. Make sure the folder structure matches the layout above.
3. Compile the project.
4. Run **Main.java**.
5. Main.java will automatically call:
   - Activity1.run()
   - Activity2.run()
   - …
   - Activity10.run()

##  Activity Structure

Each activity:
- Has its own file
- Has `package activities;`
- Contains a `public static void run()`
- Contains only code for that specific task
